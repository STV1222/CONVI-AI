# Light Frontend

This frontend contains the showcase MVP for `Light`, an AI-powered construction management platform demo.

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Validate

```bash
npm run lint
npm run build
```

## Main Files

- `pages/` demo routes
- `components/AppShell.tsx` app navigation shell
- `components/ui.tsx` reusable UI primitives
- `components/charts.tsx` Recharts components
- `data/mock.ts` rich in-memory showcase data
- `types/light.ts` shared types

## Notes

- This app is intentionally mock-data driven.
- The AI layer is simulated for demo storytelling.
- No backend is required for the current showcase flow.
